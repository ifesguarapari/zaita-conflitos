# Zaíta conflitos
Iniciativa de código aberto para um jogo narrativo inspirado no conto Zaíta esqueceu de guardar os brinquedos, de Conceição Evaristo, em que o jogador controla o irmão da personagem na tentativa de proteger a comunidade durante um confronto armado, sendo inspirado em mecânicas de defesa e estratégia em tempo real.
